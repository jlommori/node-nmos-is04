Class: Node

Node(href, services, clocks, interfacesopt)

Create a Node

Constructor

new Node(href, services, clocks, interfacesopt)

Instantiate a Node object. All params are optional, as any required items are generated automatically and optional items, such as services or api_endpoints, can be added after the node is created.
Parameters:
Name Type Attributes Description
href string HTTP access href for the Node's API (deprecated).
services Array.<Object> Array of objects containing a URN format type and href.
clocks Array.<Object> Clocks Clocks made available to Devices owned by this Node.
interfaces string | Array.<string> <optional>
 Network Interfaces made available to devices owned by this Node. Port Ids & Chassis Ids are used to inform topology discoery via IS-06 and require that interfaces implement ARP at a minimum, and ideally LLDP. Optional parameter to only include a specific interfaces or an array of interfaces. null returns all interfaces.
